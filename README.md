# ShopEase - Highly Available E-Commerce Website on AWS

## 📌 Project Overview

ShopEase is a static e-commerce website deployed on Amazon Web Services using a highly available and auto-scalable architecture.

The project demonstrates how modern cloud applications remain available during high traffic using Application Load Balancer and Auto Scaling.

---

## 🎯 Objectives

- Deploy a website on multiple EC2 instances
- Configure an Application Load Balancer
- Perform Health Checks
- Configure Auto Scaling
- Automatically launch new servers during high CPU utilization
- Demonstrate High Availability and Fault Tolerance

---

## 🛠 AWS Services Used

- Amazon EC2
- Application Load Balancer (ALB)
- Target Group
- Launch Template
- Amazon Machine Image (AMI)
- Auto Scaling Group
- Amazon CloudWatch
- Security Groups

## ⚙ Deployment Process

### Step 1

Launch two Ubuntu EC2 Instances.

### Step 2

Install Nginx.

### Step 3

Deploy ShopEase Website.

### Step 4

Create Target Group.

### Step 5

Create Application Load Balancer.

### Step 6

Verify Health Checks.

### Step 7

Create AMI.

### Step 8

Create Launch Template.

### Step 9

Create Auto Scaling Group.

### Step 10

Configure Target Tracking Scaling Policy.

### Step 11

Generate CPU Load using stress.

### Step 12

CloudWatch triggers Auto Scaling.

### Step 13

New EC2 Instance automatically launches.

### Step 14

ALB starts routing traffic to the new instance.
